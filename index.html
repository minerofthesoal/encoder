Want to contribute an encoder?
1. Use the Custom Builder tab to create and test your encoder.
2. Click Export as JSON to download the definition file.
3. Fork the repository and add your .json file to the community-encoders/ folder.
4. Add an entry to community-encoders/_index.json.
5. Open a Pull Request — once merged, your encoder will appear here automatically.
